What is protocoal?
Defination:
A protocol is a set of rules or standards that define how data is transmitted and received over a network. Protocols ensure proper communication between devices.

Types of Protocols:

1. Communication Protocol:
    - Definition: Rules that govern the exchange of data between devices.
    - Example: HTTP (HyperText Transfer Protocol) is used for transferring web pages on the internet.

2. Network Protocol:
    - Definition: Protocols that manage data transmission across networks.
    - Example: TCP/IP (Transmission Control Protocol/Internet Protocol) is used for reliable data transfer over the internet.

3. Security Protocol:
    - Definition: Protocols that ensure secure data transmission.
    - Example: SSL (Secure Sockets Layer) is used to encrypt data between web servers and browsers.

4. File Transfer Protocol:
    - Definition: Protocols used for transferring files between computers.
    - Example: FTP (File Transfer Protocol) is used to upload and download files from servers.

5. Email Protocol:
    - Definition: Protocols used for sending and receiving emails.
    - Example: SMTP (Simple Mail Transfer Protocol) is used to send emails.
