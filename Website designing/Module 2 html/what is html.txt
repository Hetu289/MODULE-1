What is HTML?

1. Full form – HTML stands for **HyperText Markup Language**.
2. Inventor – Developed by **Tim Berners-Lee** and first introduced in **1991** (not 1986).
3. Type of language – HTML is **not a programming language**; it is a **markup language** used to structure content on the web.
4. Function – HTML is interpreted by **web browsers** to display web pages.
5. Case sensitivity – HTML is **case-insensitive**, meaning `<p>` and `<P>` are treated the same.
6. File extension – HTML files use the extensions:

  
7. Example

   <p>Hello World</p>
   <P>Hello World</P>

Structure (Skeleton / Boilerplate) of an HTML Document


<!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
  <link rel="stylesheet" href="" type="text/css" />
  <link rel="icon" href="" type="image/x-icon" />
  <meta charset="utf-8" />
  <style></style>
  <script type="text/javascript"></script>
  <noscript></noscript>
</head>
<body>

</body>
</html>


What is an HTML Tag?

1. Definition – HTML tags are keywords enclosed in angle brackets `< >` that define how content is displayed on a web page.
2. Types of Tags:

   a) Paired Tags (Container Tags)

   * Have an **opening tag** and a **closing tag**.
   * The closing tag contains a forward slash `/`.
   * Example:

     
     <p>This is a paragraph</p>
     <h1>This is a heading</h1>
     

   b) Non-Paired Tags (Empty / Self-Closing Tags)

   * Have only an **opening tag**, no closing tag.
   * Used for elements that do not wrap content.
   * Example:

     <html>
     <br>      
     <hr>      
     <img src="image.jpg" alt="Image"> 
     <input type="text">  
     <meta charset="UTF-8"> 
     <link rel="stylesheet" href="style.css">
     </html>
